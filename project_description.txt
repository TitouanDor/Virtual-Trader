Gestionnaire de Portefeuille Virtuel

Objectif :
Concevoir une application web pour la gestion d'un portefeuille d'actions virtuelles en utilisant MySQL et PHP.

Description du jeu :
Chaque joueur commence avec 10 000€ pour acheter des actions virtuelles. Les actions ont un code, une description et une valeur d'achat. Certaines actions rapportent des dividendes annuels, avec une date de distribution spécifique.

Perte du jeu :
Un joueur perd si la valeur totale de son portefeuille (actions + liquidités) devient inférieure à 1000€.

Déroulement du jeu :
Le jeu fonctionne en temps pseudo-réel. Les actions des joueurs sont appliquées immédiatement. Le jeu se met à jour toutes les minutes :

-   La date du jeu avance d'un mois.
-   Les joueurs reçoivent les dividendes de leurs actions (un seul dividende par an par action).
-   Les prix des actions sont mis à jour. L'évolution du prix est calculée à partir du mois précédent, avec une variation aléatoire de -3 à +3, sans dépasser -10% et +10%, et sans descendre sous 1€.

Fonctionnalités :

-   Inscription (email, mdp)
-   Connexion
-   Modifier son mdp
-   Rechercher et suivre un joueur
-   Visualiser l'état du jeu (argent, actions, évolution du portefeuille)
-   Rechercher et filtrer les actions
-   Voir l'évolution du prix d'une action
-   Acheter/vendre une action
-   Classement des joueurs
- Dérouler les tours de jeu
